Each component has a folder containing:

1. Assets Folder
2. [compname].html - Editable file
3. [compname].htm - the compressed file
4. [compname].scss - the css file that is edited by the user
5. [compname].css - The css file that is used in preview mode
5. [compname].min.css -the css file that is publihed
6. [compname].js - the js file that is edited and used in preview mode
7. [compname].min.js the js file that is published.
