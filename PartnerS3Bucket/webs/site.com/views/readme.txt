Consider the example the home view:
Each view has a folder containing:

1. Assets Folder
2. home.html - Editable file
3. home.htm - the compressed file
4. home.scss - the css file that is edited by the user
5. home.css - The css file that is used in preview mode
5. home.min.css -the css file that is publihed
6. home.js - the js file that is edited and used in preview mode
7. home.min.js the js file that is published.
