This contains all of the configuration data, usually in json that is required for each
area solution.  So, for example:

cms config containing version number, release date etc.
voice vocab file for cms
voice vocab file for web builder
Anything else that is relevant to the cms and needs central configuration.

aboutusimg - points to either the resellers about us splash image or the vCMS splash image, depending on whether the reseller has subscribe to whitelabelling.
mode relates to cms,editor or runtime which is rt.
